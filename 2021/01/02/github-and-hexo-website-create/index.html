Github + Hexo website create guide

绑定域名：

项目主页-> Settings->GitHub Pages -> Source Branch:修改为master -> Custom domain:修改为新注册的域名，例如hexo.com

更改git默认显示分支为master：

项目主页-> Settings->Branches->Default branch:修改为master
